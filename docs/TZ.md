# Техническое задание: Calor — Android MVP

## Стек
Kotlin, Jetpack Compose, Material 3, Room, DataStore, Hilt, Navigation Compose.

## MVP
- Каталог продуктов
- Холодильник
- Блюда с расчётом ккал
- Дневник и дневной баланс
- Быстрое списание, история дня
- Настройки, экспорт/импорт JSON
- Онбординг, пустые состояния

## Вне MVP
Синхронизация, штрих-коды, виджет, графики, push, тёмная тема.
